# WineNot 🍷

**Discover wine your way.** A mobile web app that makes wine approachable for a new generation — turning winery visits into experiences worth discovering.

Built for the **European Wine Tourism Innovation Challenge** (UN Tourism, 2026).

## What it does
Three layers:
- **Discover** — a 30-second taste quiz in plain language + myth-busting content
- **Experience** — book beginner-friendly winery experiences beyond tasting
- **Belong** — a wine passport with stamps, badges and recommendations

Pilot in Greece with partner wineries Λιέπουρη (Nemea) & Νικολού; the model is replicable across every European wine region.

## Run it
It's a single static file — no build, no dependencies.
- **Live:** enable GitHub Pages (Settings → Pages → Branch: `main` → `/root`), then open the URL.
- **Local:** just open `index.html` in any browser.
